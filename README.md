# Credential Extractor

A simple Python script to scrape login form fields (e.g., username/password inputs) from a webpage.

## Installation
1. Clone the repo: `git clone https://github.com/yourusername/credential-extractor.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python extract_credentials.py`

## Usage
Modify the `url` variable in the script to target a different page.

**Note**: This is for educational purposes only. Respect website terms of service and privacy laws.
